GENERALES:

Ctrl. + E : Borra línea de codigo.
Ctrl. + S : Guarda cambios.
Shift. + F6 : RUN main proyect (actual).
Alt. + Shift + F : Le da formato al código.
Alt + Enter : En texto subrayado para ver sugerencia cuando hay error de sintaxis, import o surround.
Ctrl. + Shift + : Hace una copia de la línea de código.
Ctrl. + Space : Completa el código que estamos escribiendo.
Ctrl. + U U : Convierte el código a mayúsculas.
Ctrl. + U L : Convierte el código a minúsculas.
Alt + Insert : Agrega constructores, métodos accesotes, propiedades.
Ctrl. + G : Ir numero de línea concreto.
Ctrl. +[NOMBRECLASE] : Me despliega todas las características de la clase.
Ctrl + shift + c : comenta linea.
Alt + scroll : aumenta/disminuye pantalla.
Shift + TAB : elimina un TAB.
Alt + ENTER : importa automaticamente.
Alt + Insert : agrega constructores, getters, setters, etc.
Ctrl + mouse sobre una clase : dirije a la clase en si.

ESTRUCTURAS: 
[MOD] agregado particular tools -> options -> Code Templates.

psvm + TAB : Genera la función main.
sout + TAB : Genera System.out.println(). Con \n hace enter si necesidad de poner otro sout.
soutv + TAB : Genera System.out.println("args = " + args).
soutc + TAB : [MOD] genera System.out.print("");
soutf + TAB : [MOD] genera System.out.printf("");
jopm + TAB : [MOD] genera JOptionPane.showMessageDialog("");
jopi + TAB : [MOD] genera JOptionPane.showInputDialog(null, "");
if + TAB : Genera bloque if condicion simple.
ife + TAB : Genera bloque if condicion doble.
sw + TAB : Genera el switch.
for + TAB : genera for.
fori + TAB : Genera for para vector length.
fore + TAB : genera for reducido/ elemento.
wh/whilen/whileit + TAB : Genera los bloques while.
do + TAB :[MOD] Genera los bloques dowhile.
